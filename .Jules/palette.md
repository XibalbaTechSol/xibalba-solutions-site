## 2025-05-15 - Dynamic Accessibility for Static Sites
**Learning:** For static sites with multiple HTML files, using JavaScript to inject and manage accessibility attributes (like `aria-expanded` and `aria-controls`) is a scalable pattern. It ensures consistent behavior across all pages without requiring manual edits to every HTML file.
**Action:** When retrofitting accessibility into legacy or static sites, look for opportunities to automate attribute management via shared scripts first.
